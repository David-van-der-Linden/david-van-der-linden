Here are some public projects you could check out:
- 🎓 [QAP Thesis](https://github.com/david-van-der-linden/qap-thesis)
  - My master's thesis on the Quadratic Assignment Problem (QAP)
  - A practical (NP-Hard) problem where, despite research since 1957, some small instances are still unsolved
  - Developed a new solution method combining various mathematical techniques, validated through experiments and data analysis
- 🚚 [Markov Trucker](https://github.com/David-van-der-Linden/markov-trucker)
  - A university project using techniques that are fundamental to many AI tools in use today
- 🌲 [Advent of Code 2023](https://github.com/David-van-der-Linden/AofC2023)
  - How I solved all the AofC programming puzzles of 2023
  - [Advent of Code](https://adventofcode.com/) (AoC) is an annual programming challenge event created by Eric Wastl
